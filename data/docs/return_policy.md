# Dealer Return Policy (EU Region)

## Standard Returns
Vehicles purchased by end-customers through the dealer network are subject to a 14-day statutory withdrawal period (Widerrufsrecht) for online sales.

## B2B Returns (Dealer to Hub)
As a B2B platform, returns from Dealers to the Hub are generally **excluded**, except in cases of:
1.  **Significant Description Error**: The vehicle condition differs by more than 2 points on the condition score from the listing.
2.  **Undisclosed Damage**: Structural damage not noted in the ingredients report.
3.  **Title Issues**: Failure to provide valid registration documents within 7 days.

## Process
All claims must be submitted within 48 hours of vehicle delivery via the Claims Portal.
Compensation is capped at 5% of the transaction value for minor discrepancies.
